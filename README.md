# Registro-Chamada
Lista de chamada com as seguintes funções:
1. Inserir novos alunos;
2. Listar alunos;
3. Buscar aluno;
4. Editar aluno;
5. Remover aluno;
6. Realizar chamada, marcando alunos faltantes;
7. Ler uma data e salvar chamada em arquivo nomeado pela data.
